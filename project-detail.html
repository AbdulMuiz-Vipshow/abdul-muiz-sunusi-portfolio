case study

School Manager

A school management app built for Nigerian private schools — results, fees, attendance, and parent communication, all in one native Android app.

Java Firebase Android WhatsApp API PDF Generation

Role

Solo Developer — product, design, engineering

Timeline

[2023] — present

Platform

Native Android

Reach

[5] schools, [Kano / Northern Nigeria]

The problem

Private schools across [Northern Nigeria] were managing results, fees, and parent communication through a mix of paper records and word of mouth — slow, error-prone, and hard to scale as enrollment grew.

The solution

I built School Manager as a native Android app that lets school staff record scores, compile results, track fee payments, and share results with parents directly over WhatsApp — with automatic SMS fallback when WhatsApp delivery isn't possible.

[ screenshot — results dashboard ]

Key features

Result Compilation

Real-time progress tracking pulled directly from Firebase class records.

WhatsApp Result Sharing

Direct parent delivery with a message-credit system and SMS fallback.

Receipt Printing

WebView + PrintManager pipeline for clean, professional fee receipts.

Freemium Billing

Free plan up to 800 students; Premium plan unlocks WhatsApp sharing, unlimited students, and more.

[ screenshot — billing screen ]
[ screenshot — receipt preview ]

Business model

I designed a freemium structure to remove the friction of a mandatory subscription: a permanent Free Plan (up to 800 active students, all core features) and a Premium Plan (₦30,000/term) that unlocks WhatsApp result sharing, unlimited students, and receipt printing. A separate paid Services module — school setup, score recording, custom report card design — is fulfilled directly via WhatsApp.

Outcome

School Manager is now actively used by [5] schools, with a go-to-market strategy built around ICT champions embedded in each school and local trust-building in [Kano]. I'm currently preparing a full sales kit to scale onboarding further.
